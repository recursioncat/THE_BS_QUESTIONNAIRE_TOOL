How was This Website Made

This website was made as a tool to help people rapidly develop questionnaires. The Frontend was made with good ol' holy trinity of of HTML, CSS and Javascript(Vanilla). The Backend was developed using Flask and Python.

The core idea is to accept data from the user and process it into believable questionnaire Responses. Data entered in the questionnaire template is converted into json and sent over to python. Python has several classes for handling Questions and Options and several functions that generate and scramble data to produce a final Questionnaire. The ticks that appear in the Questionnaire are images of real hand-drawn ticks, randomised using javascript. The Graphs for the Data section are made using Matplotlib, which is a famous python module for data visualization. There is also an HTML Mapper written in python which converts object code into html which was used for generating the final pdf.

You can learn more about using this technology in in the docs page. This project is open source and you can check it out and contribute here.
